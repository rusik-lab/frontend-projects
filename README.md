# Task Manager Web Application

## Описание проекта

Данный проект представляет собой веб-приложение для управления задачами (Task Manager).  
Пользователь может зарегистрироваться, войти в систему и создавать задачи с приоритетом.  
Все задачи сохраняются в базе данных и привязаны к конкретному пользователю.

Проект реализован по архитектуре **клиент-сервер**.

Frontend написан на **Vue.js**, backend на **Node.js + Express**, база данных — **MongoDB Atlas**.

---

## Основные возможности

- регистрация пользователя
- авторизация пользователя (JWT)
- создание задач
- просмотр списка задач
- удаление задач
- установка приоритета задачи (low / medium / high)
